Arduino AI IDE Permissions

This file defines the permissions policy for the Arduino AI IDE application.

Granted Permissions:

  • Serial API - For Arduino board communication
  • USB API - For direct USB device access
  • Bluetooth - For wireless Arduino boards
  • Camera/Microphone - For AI-powered debugging features
  • Clipboard - For copy/paste functionality
  • Fullscreen - For distraction-free coding
  • Notifications - For build/upload status alerts
  • File System - For project management
  • Geolocation - For location-based Arduino projects
  • Sensors - For sensor simulation and testing

Security Notes:

  • Permissions are restricted to Arduino development use cases
  • External content is sandboxed for security
  • Only Arduino-related domains are allowed for external requests
  • All permissions can be revoked by the user at any time
